# 常见问题 FAQ

这里汇总了使用 **净影连** 时可能遇到的常见问题及其解决方法。

### Q1: 软件支持哪些格式？
基于后端强大的 `av` (FFmpeg) 库，**净影连** 支持几乎所有主流视频格式：
- `mp4`, `mkv`, `mov`, `avi`, `webm`, `flv`, `wmv`, `m4v`, `3gp`, `rmvb` 等。

### Q2: 为什么我的 GPU 加速模式无法正常工作？
目前软件的 GPU 模式专为 **NVIDIA 显卡** 的 `NVENC` 硬件加速器优化：
1. **显卡品牌**：必须是 NVIDIA。
2. **驱动程序**：请确保驱动已更新到较新版本。

### Q3: 合并时可以增加背景音乐（BGM）吗？
**净影连** 的定位是 **“整理与拼接工具”**，而非全功能的线性剪辑软件。目前专注于画面整理，暂不支持多音轨编辑和增加背景音乐。

### Q4: 输出视频的默认位置在哪里？
输出视频默认会存放在软件根目录下的 `output` 文件夹中。你也可以在高级设置中自定义输出目录。

### Q5: 合并后的视频画质会受损吗？
这取决于你选择的 **“处理模式”**：
- **均衡/质量模式** 会尽可能保持原始视频的高保真。
- 只有在 **速度优先模式** 下，为了换取极速导出，才会对画面细节进行一定程度的压缩(对于日常视频来说，损失的画面肉眼几乎不可见)。
