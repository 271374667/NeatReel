# 项目简介

**净影连 (NeatReel)** 是一款专为视频“整理控”量身打造的开源桌面工具。它并不追求复杂的剪辑功能，而是专注于解决视频合并前最琐碎、最头疼的几个环节：**去除黑边**、**校正方向**、以及**所见即所得的预览**。

有的时候下到了一些影视的剪裁或者片段，亦或是自己录制的视频分开了多个片段，想要将他们合并，有的时候横着录，有的时候竖着录，在合并的时候就麻烦了，使用 NeatReel 能帮您快速整理并且一键合并，使用质量模式还能助您在不损失画质的情况下进一步压缩体积

![主页预览](/主页_鼠标拖拽排序.png)

### 💡 为什么需要它？

在录屏、手机拍摄或整理素材时，我们常会遇到以下繁琐步骤：
1. **黑边问题**：录屏视频带着边缘黑框，直接拼接后极不美观。
2. **方向混乱**：手机竖拍和相机横拍的素材混在一起，合并后有的画面倒置，有的画面侧立。
3. **重量级工具开销**：仅仅想简单拼一下，却不得不打开庞大、沉重的专业剪辑软件。

**净影连** 旨在解决这些痛点。它将“预览、去边、校正方向、设置封面、批量合成”五个高频动作高度集成，通过多线程异步处理，让你实现真正的“一键出片”。

### ✨ 核心设计理念

- **去黑边**：智能识别，告别黑边带来的不整洁感。
- **正朝向**：方向归位，强制统一横竖屏输出。
- **一键拼**：简单配置，批量合并多段素材。
