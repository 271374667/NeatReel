# 快速上手

开启 **净影连**(NeatReel) 的第一步。

您需要先确保自己已下载并安装了 **净影连**(NeatReel)

最新的下载地址: https://github.com/271374667/NeatReel/releases/latest

### 第一步：导入视频

你可以通过以下两种方式导入你的素材：
- **拖拽导入(推荐)**：直接将文件夹或多个视频文件拖入左侧列表。
- **文件选择**：点击主界面的“添加视频”按钮。

> [!TIP]
>
> 在导入视频之前，先将需要排序的文件按照 `0001.mp4`、`0002.mp4` 这样命名，就能应用自动排序。
>
> 您也可以在 NeatReel 里面手动拖拽排序(但是这样可能会导致工作量巨大)

![主页拖放视频](/主页_拖放视频.webp)

### 第二步：校正与预览

软件右侧面板的最上方是预览屏幕，这里会展示单个视频处理后的预览画面。

1. **选中视频**：在左侧列表中点击某个视频。
2. 纠正方向：如果拍摄方向不对，点击“顺时针旋转”或“逆时针旋转”，来调整最终输出的画面方向。
3. **确认去黑边**：点击预览框下方的“使用原始视频”，可以查看没有去黑边的原始视频是什么样的，查看自动识别的黑边是否已被切除。

![主页_旋转视频](/主页_旋转视频.webp)

### 第三步：高级裁剪（可选）

如果自动去黑边的结果不完美：
1. 点击“手动裁剪”。
2. 在弹出的窗口中拖动选框。

![手动剪裁页面](/手动剪裁页面.webp)

### 第四步：输出配置

在右侧栏进行最后的设置：
1. **封面设置**：为最终成片选择一个美观的封面（可选）。
2. **横竖屏选择**：决定最终成片的比例。
3. **处理模式**：推荐选择 **“均衡模式”**。

![主页_高级设置和输出](/主页_高级设置和输出.webp)

### 第五步：开始处理

点击下方的 **“开始处理”**。
- 处理完成后，点击“打开文件夹”即可看到成果！

![输出完成](/输出完成.png)
