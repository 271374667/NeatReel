# 输出模式与硬件加速

**净影连** 将复杂的 FFmpeg 转换参数简化为三种简单直观的“处理模式”。

![输出质量选择海报](/输出质量选择海报.png)

### 🏃 速度优先 (SPEED)

- **场景**：对画质和体积没有极致要求，追求极速成片（可能会导致输出的视频体积较大）。
- **优点**：导出速度极快，CPU 占用率相对较低。

### ⚖️ 均衡模式 (BALANCED) —— 推荐

- **场景**：日常分享、短视频发布、归档。
- **优点**：在处理速度、压缩体积与画面锐利度之间找到了黄金平衡点。

### 💎 质量优先 (QUALITY)

- **场景**：高质量 Vlog、需要二次剪辑的中间素材、4K 录像合并，同时体积也是最小的。
- **优点**：最大限度保留画面原生像素细节。

### 🎮 GPU 硬件加速 (GPU)

如果你拥有一块 **NVIDIA 独立显卡** 并配置了对应的驱动支持，你可以开启 GPU 模式：
- **核心逻辑**：启用 `h264_nvenc` 硬件加速编码。
- **性能飞跃**：在高分辨率（如 4K）处理时，处理速度可提升 2-5 倍。

::: danger 注意
如果你的设备没有 NVIDIA 显卡或驱动版本过旧，开启此模式可能导致处理失败。
:::
